# CAG-tools
This tool is a simple windows console application to manage land and property gazetteer data in the [Data Transfer Format](https://www.geoplace.co.uk/documents/10181/433656/SUPERSEDED+-+DTF7+3v3-1_Third_Edition_Specification_Document_070212-FINAL.pdf/d38ad3ac-9f9d-4121-bfd6-85204bfc922f?version=1.1) (DTF 7.3). 

Records of type 15 or 24 as defined in the specification are extracted to separate files unchanged.
